<!-- GETTING STARTED -->
## Installation

To build the app locally, follow those basic steps:

1. Clone the repo
   ```sh
   git clone https://github.com/Mendzel/Portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the app locally
   ```sh
   npm run nuxt dev
   ```
